# 极值

## 一、一元函数的极值

### 1.1 极值的定义

设函数 $f(x)$ 在点 $x_0$ 的某邻域内有定义，如果对于该邻域内的任意点 $x$（除 $x_0$ 外），都有：

- **极大值**：$f(x) \leq f(x_0)$，则称 $f(x_0)$ 为函数 $f(x)$ 的一个**极大值**，$x_0$ 为**极大值点**
- **极小值**：$f(x) \geq f(x_0)$，则称 $f(x_0)$ 为函数 $f(x)$ 的一个**极小值**，$x_0$ 为**极小值点**

极大值和极小值统称为**极值**，极大值点和极小值点统称为**极值点**。

> **注意**：极值是局部概念，只在点的邻域内有意义。

### 1.2 极值的判别法

#### 定理一：费马定理（Fermat）

若函数 $f(x)$ 在点 $x_0$ 处可导，且在 $x_0$ 处取得极值，则必有：
$$f'(x_0) = 0$$

**逆命题不成立**：$f'(x_0) = 0$ 只是极值的**必要条件**，而非充分条件。

#### 定理二：一阶导数判别法

设函数 $f(x)$ 在 $x_0$ 的某邻域内可导（$f'(x_0)$ 可以不存在，但连续）：

1. 当 $x < x_0$ 时，$f'(x) > 0$；当 $x > x_0$ 时，$f'(x) < 0$（导数由正变负）$\Rightarrow$ $x_0$ 为**极大值点**
2. 当 $x < x_0$ 时，$f'(x) < 0$；当 $x > x_0$ 时，$f'(x) > 0$（导数由负变正）$\Rightarrow$ $x_0$ 为**极小值点**
3. 若 $f'(x)$ 在 $x_0$ 两侧同号，则 $x_0$ **不是极值点**

#### 定理三：二阶导数判别法

设函数 $f(x)$ 在 $x_0$ 处二阶可导，且 $f'(x_0) = 0$：

- 若 $f''(x_0) < 0$ $\Rightarrow$ $x_0$ 为**极大值点**
- 若 $f''(x_0) > 0$ $\Rightarrow$ $x_0$ 为**极小值点**
- 若 $f''(x_0) = 0$ $\Rightarrow$ 判别法失效，需用其他方法

#### 定理四：高阶导数判别法

若 $f'(x_0) = f''(x_0) = \cdots = f^{(n-1)}(x_0) = 0$，而 $f^{(n)}(x_0) \neq 0$：

- $n$ 为奇数：$x_0$ **不是极值点**
- $n$ 为偶数：$x_0$ 是极值点，且 $f^{(n)}(x_0) < 0$ 时为极大值，$f^{(n)}(x_0) > 0$ 时为极小值

## 二、二元函数的极值

### 2.1 极值的定义

设函数 $z = f(x, y)$ 在点 $(x_0, y_0)$ 的某邻域内有定义，如果对于该邻域内的任意点 $(x, y)$（除 $(x_0, y_0)$ 外），都有：

- **极大值**：$f(x, y) \leq f(x_0, y_0)$ $\Rightarrow$ $(x_0, y_0)$ 为**极大值点**
- **极小值**：$f(x, y) \geq f(x_0, y_0)$ $\Rightarrow$ $(x_0, y_0)$ 为**极小值点**

### 2.2 极值的判别法

#### 定理：二元函数极值的必要条件

若函数 $f(x, y)$ 在点 $(x_0, y_0)$ 处偏导数存在且取得极值，则：
$$f_x(x_0, y_0) = 0, \quad f_y(x_0, y_0) = 0$$

#### 黑塞矩阵法（正定性判别）

设函数 $f(x, y)$ 在点 $(x_0, y_0)$ 的某邻域内二阶连续偏导，且：
$$f_x(x_0, y_0) = 0, \quad f_y(x_0, y_0) = 0$$

构造**黑塞矩阵**（Hessian Matrix）：
$$H = \begin{pmatrix} f_{xx} & f_{xy} \\ f_{yx} & f_{yy} \end{pmatrix}\Bigg|_{(x_0, y_0)}$$

根据黑塞矩阵的正定性判别：

| 黑塞矩阵 $H$ 的性质 | 结论 |
|------|------|
| **正定**（所有特征值 $> 0$） | $(x_0, y_0)$ 为**极小值点** |
| **负定**（所有特征值 $< 0$） | $(x_0, y_0)$ 为**极大值点** |
| **不定**（特征值异号） | $(x_0, y_0)$ 为**鞍点** |
| **半正定/半负定**（有零特征值） | 判别法失效，需进一步分析 |

记 $A = f_{xx}$，$B = f_{xy}$，$C = f_{yy}$，则：
- 正定 $\Leftrightarrow$ $A > 0$ 且 $AC - B^2 > 0$
- 负定 $\Leftrightarrow$ $A < 0$ 且 $AC - B^2 > 0$
- 不定 $\Leftrightarrow$ $AC - B^2 < 0$

> **几何意义**：极大值点像山顶，极小值点像山谷，鞍点像马鞍的中心点。

#### 二阶全微分法

黑塞矩阵的判别等价于二阶全微分的符号判别：
$$d^2f = f_{xx}dx^2 + 2f_{xy}dxdy + f_{yy}dy^2$$

- $d^2f > 0$（正定）$\Rightarrow$ 极小值
- $d^2f < 0$（负定）$\Rightarrow$ 极大值
- 符号不定 $\Rightarrow$ 鞍点

## 三、条件极值

### 3.1 条件极值的定义

求函数 $z = f(x_1, x_2, \cdots, x_n)$ 在 $m$ 个约束条件
$$\begin{cases} g_1(x_1, \cdots, x_n) = 0 \\ g_2(x_1, \cdots, x_n) = 0 \\ \vdots \\ g_m(x_1, \cdots, x_n) = 0 \end{cases}$$
下的极值，称为**条件极值**，其中 $m < n$。

### 3.2 拉格朗日乘数法

#### 基本原理

构造**拉格朗日函数**：
$$\mathcal{L}(x_1, \cdots, x_n, \lambda_1, \cdots, \lambda_m) = f(x_1, \cdots, x_n) + \sum_{j=1}^{m} \lambda_j g_j(x_1, \cdots, x_n)$$

其中 $\lambda_1, \lambda_2, \cdots, \lambda_m$ 称为**拉格朗日乘子**。

#### 求解步骤

1. 构造拉格朗日函数 $\mathcal{L}$
2. 对所有变量（含乘子）求偏导并令其为 $0$：
$$\begin{cases} \dfrac{\partial \mathcal{L}}{\partial x_i} = \dfrac{\partial f}{\partial x_i} + \sum_{j=1}^{m} \lambda_j \dfrac{\partial g_j}{\partial x_i} = 0 \quad (i = 1, 2, \cdots, n) \\[8pt] g_j(x_1, \cdots, x_n) = 0 \quad (j = 1, 2, \cdots, m) \end{cases}$$
3. 解上述 $n + m$ 个方程组成的方程组，得到候选点
4. 结合实际问题判断是否为极值点

#### 二元函数单约束的特例

当 $n = 2$，$m = 1$ 时：
$$\mathcal{L}(x, y, \lambda) = f(x, y) + \lambda g(x, y)$$
方程组简化为：
$$\begin{cases} f_x + \lambda g_x = 0 \\ f_y + \lambda g_y = 0 \\ g(x, y) = 0 \end{cases}$$

#### 二阶充分条件（单约束情形）

对于 $z = f(x, y)$ 在约束 $g(x, y) = 0$ 下的极值，构造：
$$\mathcal{L}(x, y, \lambda) = f(x, y) + \lambda g(x, y)$$

计算二阶偏导数，记：
$$\bar{A} = \mathcal{L}_{xx} = f_{xx} + \lambda g_{xx}$$
$$\bar{B} = \mathcal{L}_{xy} = f_{xy} + \lambda g_{xy}$$
$$\bar{C} = \mathcal{L}_{yy} = f_{yy} + \lambda g_{yy}$$

在约束曲线的切向方向上：
- 若 $\bar{A}\bar{C} - \bar{B}^2 > 0$ 且 $\bar{A} > 0$ $\Rightarrow$ **极小值**
- 若 $\bar{A}\bar{C} - \bar{B}^2 > 0$ 且 $\bar{A} < 0$ $\Rightarrow$ **极大值**

#### 几何意义

拉格朗日乘数 $\lambda_j$ 的物理意义：表示目标函数对第 $j$ 个约束的**敏感程度**，即约束边界变化时目标函数的边际变化率。

### 3.3 不等式约束的极值（KKT 条件）

#### 问题描述

求函数 $z = f(x_1, \cdots, x_n)$ 在不等式约束
$$g_j(x_1, \cdots, x_n) \leq 0 \quad (j = 1, 2, \cdots, m)$$
下的极值。

> **注**：$g_j(x) \geq 0$ 的约束可写成 $-g_j(x) \leq 0$。

#### KKT 条件（Karush-Kuhn-Tucker）

构造拉格朗日函数：
$$\mathcal{L}(x, \lambda) = f(x) + \sum_{j=1}^{m} \lambda_j g_j(x)$$

极值点 $x^*$ 需满足以下条件：

1. **平稳性条件**：
$$\nabla f(x^*) + \sum_{j=1}^{m} \lambda_j \nabla g_j(x^*) = 0$$

2. **原始可行性**：
$$g_j(x^*) \leq 0 \quad (j = 1, \cdots, m)$$

3. **对偶可行性**（乘子非负）：
$$\lambda_j \geq 0 \quad (j = 1, \cdots, m)$$

4. **互补松驰条件**：
$$\lambda_j g_j(x^*) = 0 \quad (j = 1, \cdots, m)$$

#### 互补松驰条件的含义

- 若 $g_j(x^*) < 0$（约束未激活）$\Rightarrow$ $\lambda_j = 0$
- 若 $g_j(x^*) = 0$（约束在边界上激活）$\Rightarrow$ $\lambda_j \geq 0$ 可以非零

#### 几何解释

- **有效约束**（active constraints）：$g_j(x^*) = 0$ 的约束在边界上
- **无效约束**（inactive constraints）：$g_j(x^*) < 0$ 的约束未触及边界，对解无影响

#### 例子

求 $f(x, y) = x^2 + y^2$ 在 $x + y \leq 1$ 下的极小值。

解：构造 $\mathcal{L} = x^2 + y^2 + \lambda(x + y - 1)$

KKT 条件：
- $2x + \lambda = 0$
- $2y + \lambda = 0$
- $\lambda \geq 0$
- $x + y - 1 \leq 0$
- $\lambda(x + y - 1) = 0$

若 $\lambda = 0$：解为 $(0, 0)$，但 $0 + 0 \leq 1$ 满足，此时 $f = 0$。

若 $x + y = 1$：由前两式得 $x = y = -\lambda/2$，代入得 $-\lambda = 1$，即 $\lambda = -1 < 0$（不满足 $\lambda \geq 0$），故此情况无效。

因此最优解为 $x^* = 0, y^* = 0$，极小值 $f(0, 0) = 0$。

## 四、思维导图

```
极值
├── 一元函数极值
│   ├── 定义：局部最大/最小
│   ├── 必要条件：f'(x₀)=0（费马定理）
│   └── 判别法
│       ├── 一阶导数法（符号变化）
│       ├── 二阶导数法（f''(x₀)正负）
│       └── 高阶导数法（首个非零导数阶数奇偶）
└── 二元函数极值
    ├── 定义：局部最大/最小
    ├── 必要条件：fₓ=0, fᵧ=0
    └── 判别法：黑塞矩阵
        ├── AC-B²>0, A>0 → 极小
        ├── AC-B²>0, A<0 → 极大
        ├── AC-B²<0 → 鞍点
        └── AC-B²=0 → 待定
```
